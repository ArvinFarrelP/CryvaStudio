Installation & Usage

Get started with CipherVision in minutes

Install Dependencies 
# Install required Python packages
pip install matplotlib numpy ttkbootstrap
Copied!
Build Executables (Windows) 
# Install PyInstaller
pip install pyinstaller

# Build standalone executables
python -m PyInstaller --onefile --windowed frequency_analysis.py
python -m PyInstaller --onefile --windowed encrypt_text.py

# Output location
dist/
  frequency_analysis.exe
  encrypt_text.exe

# Run
dist/frequency_analysis.exe
Copied!
Usage from source 
# Encrypt text
python encrypt_text.py

# Run frequency analysis
python frequency_analysis.py
Copied!
