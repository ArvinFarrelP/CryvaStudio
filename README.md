# CipherVision Labs

Cryptography Research & Cipher Intelligence Lab

License: MIT | Version: 1.0.0 | Platform: Windows

Research-driven cryptography lab building open-source tools for classical cipher analysis, frequency-based decryption, and security education.

---

Featured Project: CipherVision

A complete cryptographic analysis workbench for Windows.

Features:

- Frequency Analysis Engine
- Caesar Brute-Force Solver (all 26 shifts)
- Manual Cipher Mapping with live preview
- Histogram Visualization
- Save & Load Sessions
- Fully Offline standalone EXE

Tech Stack: Python | matplotlib | tkinter | PyInstaller

---

Download: https://github.com/ArvinFarrelP/CipherVision/releases/tag/v1.0.0

Research Focus:

- Statistical cryptanalysis of classical cipher families
- Letter frequency distribution across languages
- Automated decryption algorithms
- Educational cryptography tooling
